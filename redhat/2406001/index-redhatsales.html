多功能 A 廳基礎班 - 101 Red Hat Ansible workshop

如果你從未嘗試過 Red Hat Ansible Automation Platform，歡迎報名參加體驗。
本工作坊適合那些想要了解 Ansible 自動化的基礎知識，並了解工作原理以及如何運用於自動化任務。
此外，將提供紅帽產品與上游專案之間的差異資訊。

如果你對 Ansible 自動化還不熟悉，本工作坊將提供分為兩大部分教學：



您將編寫您的第一個 Ansible Playbook，進行 Jinja 模板的編寫，並操作高級別的 Ansible 角色 (Roles)。接下來，您將開始進行自動化控制器 (automation controller)，了解庫存和憑證管理、項目、任務模板、調查、工作流程等。

完成本實驗操作後，您將準備好開始使用 Ansible 自動化平台來滿足您的自動化需求。

多功能 B 廳進階班 - Red Hat Event-Driven Ansible workshop

作為 Red Hat ® Ansible ® Automation Platform 的一部分，事件驅動的 Ansible (Event-driven Ansible, EDA) 能夠智能處理 IT 環境符合指定條件的離散事件，採取對事件最適當的回應，執行自動化操作以解決或補救事件。本工作坊將演示事件驅動的 Ansible 如何提供所需的事件處理功能，以自動化節約耗時的任務並應對任何 IT 領域中的變化因素。

在本工作坊中，我們將介紹 EDA 的基礎知識，包括原始碼、規則和操作等基本概念，以及實際操作 Ansible Rulebook CLI、Rulebooks、Source Plugins 和 EDA content collections 等技術。
本工作坊要求學生具備基礎的 Ansible 命令行 (Command-line Ansible)、Visual Studio Code 和 Git 知識。

報名 EDA 實機操作工作坊，了解事件驅動的 Ansible 如何幫助您自動化攻克 IT 環境的各種任務。

高級/進階 Ansible 用戶和 SRE 推薦參加。
本工作坊介紹了可觀測性 (observability) 和自動化領域。

注意事項

  本實機操作體驗工作坊為進階課程
  學員建議具有構建 Ansible Playbook 和使用 AAP 的經驗
  基本理解 GIT 和 Source Control

  必須攜帶 / 使用安裝了 Chrome 73+、Firefox 60+、Edge 40+ 或 Safari 12+ 的筆記型電腦
